Coin Tracker

It shows the information of cryptocurrency. If you click a name of the cryptocurrency, the page will show you all about the cryptocurrency, by explaining about the coin, and comparing the price of coin, and visualizing the price per each time or day in line chart/candle stick chart. It fetches data from cryptocurrency API by using React Query, and presents the information in real time in the page. The light and dark mode is built with Styled Components and Recoil. The charts are built with Apex Chart.
